딕셔너리(Dictionary)

  • 앞쪽_화살표: 문법 기호/함수 : {key1 : value1, key2 : value2, key3 : value3 ... } / 함수 : dict()
  • 앞쪽_화살표: key로 가질 수 있는 데이터 타입 : 문자, 숫자, 튜플(Tuple)
  • 앞쪽_화살표: value로 가질 수 있는 데이터 타입 : 문자, 숫자, 리스트(List), 딕셔너리(Dict) ...
  • 앞쪽_화살표: 아이템 = key&value 하나의 쌍
  • 앞쪽_화살표: 종류
    • 아이템이 없는 딕셔너리
    • 1차원 딕셔너리
    • 다차원 딕셔너리
  • 앞쪽_화살표: 특징
    • key, value의 차이점
    • 인덱스 번호를 가지지 않는다.
